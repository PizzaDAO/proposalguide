üçï Ingredient #2: Create Your Proposal

Connect to the PizzaDAO Poll Creator and follow these 12 steps:

  1. Enter Poll Title: Type in the name of your event. E.g: ETHDenver Pizza Party
  2. Select Channel: Select the "proposals" channel.
  3. Fill Poll Options: Aye, Nay, Abstain
  4. Enter Description:
    Fill the description field following the template provided below. If it's not fit your proposal, check here for more templates.
  5. Leave "allow anyone to add poll option" deselected
  6. Select Mentions: Tag yourself, your proposal sponsor (if applicable), and Don Scrimmy John. In 'Roles', tag @Super Voter.
  7. Keep "Single Vote Per User" selected
  8. Select "Anonymous"
  9. Enter End Time: Fill field with: "In 7 Days"
  10. Enter Image:
    Insert link to image here. Flyer images can be powerful to make your proposal stand out!
  11. Select Role Restrictions:
    Enter 'Box Mafia' & 'Pizza Mafia'.
    This is very important as we only allow our box and pizza NFT holders the ability to vote.
  12. Submit with the "Create Poll" button
